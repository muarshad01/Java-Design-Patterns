## How to Install Java

Search [Java Downloads](https://www.oracle.com/java/technologies/downloads/)

```
$ java --version
$ /usr/libexec/java_home -V 
```

## Export JAVA_HOME & PATH

```
export JAVA_HOME=/usr/libexec/java_home
export PATH=$PATH:$JAVA_HOME/bin
```
